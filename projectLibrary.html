CASE STUDY: Williston Park Library

Building Inclusive Digital Access

Background

This project focused on improving accessibility across the Home, Events, and Contact pages of the Williston Park Public Library website. As part of my senior capstone, I redesigned these pages to meet WCAG 2.2 AA standards and ensure a user-friendly experience for all users, including individuals who rely on screen readers or keyboard-only navigation.

Challenge

The original site contained accessibility issues such as missing alt text, unlabeled form inputs, low-contrast colors, incorrect ARIA roles, and non-semantic markup. These barriers made navigation difficult for blind and low-vision users and prevented WCAG compliance.

Process

I began with an accessibility audit using Lighthouse, WAVE, and manual keyboard + screen-reader testing. I rebuilt core structures to improve semantics, operability, and meaningful feedback.

  • Implemented semantic landmarks and a logical heading hierarchy.
  • Added descriptive alt text and clarified link labels.
  • Improved color contrast and added visible focus outlines.
  • Rebuilt the calendar with proper table headers and a list-view fallback.
  • Corrected ARIA roles and ensured proper form label associations.
  • Replaced the interactive map with an accessible static alternative.

Results

All redesigned pages reached full WCAG 2.2 AA compliance. Lighthouse scores increased from 75 to 100 across all pages. Keyboard-only navigation now functions smoothly, and users receive consistent, perceivable feedback throughout the interface.
